# Domus Studio — потолочные решения

Локальное приложение для замера помещений, проектирования потолка, сметы и подготовки производственных файлов.

## Файлы

- `index.html` — интерфейс приложения.
- `styles.css` — адаптивные стили.
- `app.js` — геометрия, каталог, смета и экспорт.
- `cloud-config.example.json` — заготовка настроек будущего API без секретов.

## Локальный режим

Откройте `index.html` в браузере. Проекты сохраняются в `localStorage`; для передачи на производство используйте DXF, CSV и производственный JSON-пакет.

## Облачный режим

Перед публикацией нужно выбрать сервер, заполнить `cloud-config.example.json`, вынести проекты из `localStorage` в API и добавить серверную авторизацию. Секреты не должны храниться в клиентском JavaScript.
